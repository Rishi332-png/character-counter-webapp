Charcater Counter

This is a character counter web application in this you can customize the size of the paragraph when put in it and you can check the actual space and number od characters the paragraph is taking . Just enter the number of characters you and number of lines if needed and enter the content in the below box on hitting enter the it will display the extra space present if the content is upto the mark then it will show the exct number .
